# SDESAWEB25
# Web Asistente de Configuración de PCs
---

## Descripción del proyecto
Página web enfocada a ayudar a personas sin conocimientos de informática a seleccionar **componentes compatibles para un PC**, mostrando recomendaciones, avisos y explicaciones sencillas durante todo el proceso. En el enlace se podrá observar una página algo parecida a lo que quiero intentar hacer o proponer.

[PcPartPicker](https://es.pcpartpicker.com)
---

## Público objetivo
Usuarios que **necesitan montar** o comprar un ordenador y no saben qué piezas son compatibles entre sí o cuál es la mejor opción para su presupuesto.

---

## Funcionalidades principales
- Sistema de selección guiada paso a paso
- Filtro automático por compatibilidad
- Información explicada en lenguaje sencillo
- Opiniones sobre configuraciones creadas por usuarios
- Guardado de configuraciones

## Pasos de uso 
1. Entrar en la aplicación web
2. Seleccionar necesidades y presupuesto
3. Elegir componentes sugeridos
4. Guardar la configuración final
5. Consultar opiniones de otros usuarios
