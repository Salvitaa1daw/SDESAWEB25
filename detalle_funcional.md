## Detalle de la Funcionalidad: Selección Guiada de Componentes

La funcionalidad de **selección guiada de componentes** es uno de los pilares principales del proyecto.  
Su objetivo es ofrecer a los usuarios una experiencia sencilla, clara y progresiva para elegir las piezas de su futuro ordenador, incluso si no tienen conocimientos técnicos.

A lo largo de este flujo, la plataforma proporciona información útil, avisos de compatibilidad y recomendaciones basadas en parámetros como presupuesto, tipo de uso y preferencias del usuario.

La idea central es permitir que cualquier persona pueda configurar un PC funcional sin necesidad de saber detalles complejos sobre hardware, reduciendo así errores frecuentes y la *frustración* habitual en este tipo de procesos.

### Pasos principales del flujo
1. El usuario indica el uso principal del equipo (ofimática, gaming, edición, etc.).
2. La aplicación sugiere una serie de componentes base adaptados a ese perfil.
3. El usuario selecciona cada componente de una lista filtrada por compatibilidad.
4. El sistema muestra advertencias en caso de incompatibilidad.
5. Se genera una configuración final lista para guardar o compartir.

### Conceptos clave incluidos
- Elección guiada paso a paso mediante interfaz intuitiva.  
- Compatibilidad verificada entre CPU, placa base, RAM y demás piezas.  
- Recomendaciones automáticas según el perfil del usuario.
